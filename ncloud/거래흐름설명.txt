1. 사용자가 브라우저에서 https://compvalue.xyz 접속

2. Cloudflare Tunnel (cloudflared) → 로컬 127.0.0.1:80
    - ~/.cloudflared/config.yml

3. Minikube Ingress Controller (Ingress-Nginx)
    - 80 포트 점유중

4. Ingress Controller → Ingress Rule 확인

5. Ingress → Kubernetes Service

6. 서비스 → 파드 (애플리케이션)

7. 응답 경로 (역방향)

-------------------------------

M1 Mac 이라 NodePort 로 외부접근이 막혀서 터널로 접근

클라우드플레어 터널 실행 명령어 (로컬 PC에 nginx 안해줘도 됨)
- cloudflared tunnel run compvalue-tunnel

미니큐브 터널 실행(인그레스) : 미니큐브 포트포워딩 안해도됨
- minikube -p compvalue tunnel
확인
- kubectl -n ingress-nginx get svc ingress-nginx-controller -o wide
